# Руководство по внесению вклада

Спасибо за интерес к проекту EyeCare Reminder!

## Как внести вклад

1. Fork репозиторий
2. Создайте ветку для вашей функции (`git checkout -b feature/AmazingFeature`)
3. Внесите изменения
4. Закоммитьте изменения (`git commit -m 'feat: добавлена AmazingFeature'`)
5. Запушьте в ветку (`git push origin feature/AmazingFeature`)
6. Откройте Pull Request

## Стиль кода

- Следуйте PEP 8
- Добавляйте docstrings для всех функций
- Используйте type hints где возможно
- Пишите понятные commit messages

## Commit Messages

Используйте формат Conventional Commits:
- `feat:` для новых функций
- `fix:` для исправлений багов
- `docs:` для документации
- `refactor:` для рефакторинга
- `test:` для тестов
